# 144. 二叉树的后序遍历

### 题目描述（难度：中等）

- 给定一个二叉树，返回它的 后序 遍历。

- 示例 1:

```
输入: [1,null,2,3]
   1
    \
     2
    /
   3

输出: [3,2,1]
```

### 思路 1:

- 颜色标记法(迭代法, 模拟系统栈)
  - 将访问过的元素标记为灰色, 未访问过的元素标记为白色;
    从栈顶取出访问元素:
    若为灰色元素, 则打印之;
    若为白色元素, 按照中 -> 右 -> 左的顺序推入栈, 同时将白色元素标记为灰色元素;

### 代码 1

```js
/**
 * Definition for a binary tree node.
 * function TreeNode(val) {
 *     this.val = val;
 *     this.left = this.right = null;
 * }
 */
/**
 * @param {TreeNode} root
 * @return {number[]}
 */
var postorderTraversal = function (root) {
  let printArr = [];
  if (!root) return [];
  let stack = [];
  stack.push({
    color: "white",
    node: root,
  });

  while (stack.length > 0) {
    let topVal = stack.pop();
    const { color, node } = topVal;
    if (color === "gray") {
      printArr.push(node.val);
    } else {
      stack.push({ color: "gray", node });
      node.right && stack.push({ color: "white", node: node.right });
      node.left && stack.push({ color: "white", node: node.left });
    }
  }
  return printArr;
};
```

### 思路

- 递归

### 代码 3

```js
/**
 * Definition for a binary tree node.
 * function TreeNode(val) {
 *     this.val = val;
 *     this.left = this.right = null;
 * }
 *
/**
 *
 * @param {TreeNode} root
 * @return {number[]}
 */
var preorderTraversal = function (root) {
  if (root) {
    return [
      ...preorderTraversal(root.left),
      ...preorderTraversal(root.right),
      root.val,
    ];
  } else {
    return [];
  }
};
```
